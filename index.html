Turm und Wächter - FEN Viewer

Spiel beendet!

Figurenerklärung:

  • r1, r2, r3 = roter Soldat mit Turmhöhe 1-3
  • RG = roter Wächter (Guard)
  • b1, b2, b3 = blauer Soldat mit Turmhöhe 1-3
  • BG = blauer Wächter (Guard)

Java FEN-Format:

  • Zahlen (z.B. 3) = Anzahl leerer Felder
  • RG/BG = Wächter (rot/blau)
  • r1/b1 = Türme mit Höhenangabe
  • / = Trennt Reihen
  • r/b am Ende = Spieler am Zug
